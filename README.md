# Parature Analytics
## Description
Single page application for viewing Parature reporting data.

## Installation
In `config.py` update:
* SECRET_KEY

Install the package locally (for use on our system), with:
```
python setup.py install
```

Or

Create a source distribution with:

```
python setup.py sdist
```

This will create `dist/paratureanalytics-0.5.tar.gz` inside the top-level directory.
